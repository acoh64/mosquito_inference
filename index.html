Predicting Mosquito Flight Behavior using Bayesian Learning

These simulations are models learned from 3D tracking experiments of mosquitoes. The model is converted to a 2D model for the purpose of visualization. Each panel shows a different experimental condition, allowing comparison of how mosquitoes behave in different conditions. To use an interactive version of the site, check out: mosquitoapp-production.up.railway.app
